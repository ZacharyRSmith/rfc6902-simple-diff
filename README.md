rfc6902 diff generator
======================

Another json diff tool.

If you are looking for a whole implementation(differ and patcher), please check @chbrown's [rfc6902](https://www.npmjs.org/package/rfc6902).

Usage
-----

```shell
npm install rfc6902-diff
```

### Develop

```shell
npm install
npm run dev
```

### Test

```shell
npm install
npm run test
```
